## Entfernung Des Calibre-Plugins - DRM Befreien: Schritt-Für-Schritt-Anleitung Für 2024

## DRM mit CALIBREs Plugin entfernen 2024

Posted by [Ada Wang](https://plus.google.com/+AdaWang/posts) on 7/24/2024 10:37:19 AM.

4.8 [(25 comments)](http://www.epubor.com/#comment-area) 



![follow](http://www.epubor.com/images/follow.png)

Das Calibre DRM Removal Plugin, bekannt als Alfs DeDRM Plugin Tool, wurde entwickelt, um DRM aus eBooks, einschließlich Kindle, Kobo und Adobe DRM, zu entfernen. Es besteht kein Zweifel, dass der Erfolg von Calibre hauptsächlich von diesen Plugins abhängt, und die DRM Removal-Plugins von Alf sind die Superstars unter ihnen. Aber viele Leute sagen, dass die Benutzeroberfläche und die Benutzererfahrung des Calibre nicht so freundlich für technische Neulinge sind. Keine Erwähnung der Plugin-Installation, die eine weitere Herausforderung für sie wäre. Um Ihnen die reibungslose Verwendung dieses kostenlosen DRM-Entfernungs-Plugins zu erleichtern, werden in diesem Artikel die beiden folgenden Probleme behandelt: Installieren des DRM-Entfernungs-Plugins auf Calibre unter Windows / Mac und Entfernen von eBook DRM mit Calibre DRM-Entfernungs-Plugins.

1Installation von Calibre

Das Herunterladen und Verwenden von [Calibre](http://calibre-ebook.com/download) ist völlig kostenlos. Klicken Sie auf die Schaltflächen unten, um die Windows- und Mac-Versionen direkt herunterzuladen.

3Installation der Plugins für CALIBRE

Führen Sie Calibre aus, klicken Sie in der oberen Menüleiste auf "Einstellungen" und sehen Sie sich unten im neuen Fenster "Erweiterungen".

![calibre einstellungen](https://epubor.com/images/uppic/calibre-einstellungen.png)

(Keine Notwendigkeit, es zu entpacken.) 

Klicken Sie auf Plugins und dann auf "Erweiterung aus Datei laden".

![Erweiterung aus Datei laden](https://epubor.com/images/uppic/erweiterung-aus-datei-laden.png)

Suchen Sie das DeDRM-Plugin, das Sie in Schritt 2 heruntergeladen haben, und klicken Sie auf "Öffnen". Das Kaliber warnt Sie vor dem Sicherheitsrisiko. Klicken Sie auf "Ja", um fortzufahren.

![Calibre Sicherheitsrisiko](https://epubor.com/images/uppic/sicherheitsrisiko.png)

Schließlich können Sie die Erfolgreich sehen.

![deDRM erfolgreich](https://epubor.com/images/uppic/dedrm-dateityp-erweiterungen.png)

Zurück zum Plugin Center sehen Sie die Plugin-Informationen unter "Dateityp-Erweiterungen".

![calibre Plugin entfernen](https://epubor.com/images/uppic/calibre-plugin-entfernen.png)

43 Häufigste Lösungen für das Entfernen von Fehlern durch Calibre DRM

Ist es häufig der Fall, dass Sie DRM nicht aus E-Books entfernen können? obwohl das Calibre DeDRM-Plugin erfolgreich installiert wurde. Hier zeige ich Ihnen einige gängige Lösungen, mit denen die meisten Probleme gelöst werden können.

Problemumgehung 1 Laden Sie das eBook neu in Calibre.

In den meisten Fällen konnten die Benutzer das DRM nicht entfernen, da sie das Buch vor der Installation des Plugins hinzugefügt haben. Durch einfaches erneutes Laden des Buches kann dieses Problem nicht behoben werden. Folgendes müssen Sie tun:

Schritt 1\. Wählen Sie ein E-Book in der Calibre-Bibliothek aus und klicken Sie mit der rechten Maustaste auf ? „Buch löschen“ ? „Ausgewähltes Buch löschen“.

Schritt 2\. Starten Sie Calibre neu und fügen Sie es erneut zur Calibre-Bibliothek hinzu.

Problemumgehung 2 Aktualisieren Sie das Calibre DeDRM Plugin.

Manchmal schlägt das Plugin fehl, nur weil Sie die ältere Version verwenden. In diesem Fall können Sie die folgenden Schritte ausführen, um Calibre DeDRM erneut zu aktualisieren.

Schritt 1\. Führen Sie Calibre aus und klicken Sie auf Einstellungen.

Schritt 2\. Suchen Sie das Plugin unten im Popup-Fenster und geben Sie DeDRM in das Suchfeld ein.

Schritt 3\. Suchen Sie die alte Version des Plugins auf der Registerkarte Plugins für Dateitypen. und klicken Sie auf Plugin entfernen, um die alte Version des Plugins zu entfernen. Klicken Sie dann auf Übernehmen.

Schritt 4\. Kaliber schließen. und laden Sie die neueste Version 6.8.0 des DeDRM-Plugins herunter.

Schritt 5\. Führen Sie Calibre erneut aus. Klicken Sie in den Einstellungen > Plugin auf Plugin aus Datei laden und fügen Sie das neue DeDRM-Plugin zu Calibre hinzu.

Schritt 6\. Schließen Sie Calibre und starten Sie es neu.
