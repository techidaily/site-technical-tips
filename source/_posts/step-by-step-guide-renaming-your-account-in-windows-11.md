### Key Takeaways

* To change the user name of a Microsoft account, log in to your Microsoft Account, click on your name, and then click "Edit name."
* To change the user name of a local account, use Control Panel. Click Start, type "Control Panel," click "User Accounts," and then click "Change your account name."
* Alternatively, you can change the user name using netplwiz. Press Windows+R, type "netplwiz," select the user name you want to change, click "Properties," and then enter the new user name.

 If you want to change the user name associated with your Windows user account, you can — but there are different ways to change it depending on what kind of account you have. Here's how you do it.

##  How to Change a Microsoft Account Name

 If you're signed in with a Microsoft account you have to change your user name on the Microsoft website.

 To begin, [log in to your Microsoft Account](https://login.live.com/). Edge users will probably be logged in automatically. If you're not using Microsoft Edge as your default browser, you'll have to log in manually. You can also navigate to Settings > Accounts > Your Info and click "Manage My Account" to access the page.

 Once you've logged in, click on your name at the top of the screen.

![Click your name](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2022/02/click-your-name.png) 

 Then look towards the bottom-right side of the first section — click "Edit name."

![Click "edit name."](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2022/02/click-edit-name.png) 

 Fill in your desired user name and the captcha in the popup, then click "Save."

![Fill out your name and captcha; then click "Save."](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2022/02/fill-out-first-name-last-name-captcha-then-click-Save.png) 

##  How to Change a Local Account Name with Control Panel

 The way you change a user name is different if you're logged in with a local account. You'll need to use Control Panel, since the option isn't available in the Settings App.

 First, click Start. Then, type "Control Panel" into the search bar and hit Enter.

 The user interfaces in Windows 10 and Windows 11 are not identical, though they are similar. Within Control Panel and subsequent menus, the differences are only cosmetic. If something looks a little different, don't worry about it.

 Click "User Accounts" in the Control Panel.

![Click &quot;User Accounts.&quot;](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2022/02/click-user-accounts.png)

##  How to Change a Local Account Name With netplwiz

 Alternatively, you can change your user name using netplwiz. The netplwiz method works on both Windows 10 and Windows 11.

 To start, hit Windows+r and type "netplwiz" into the run box, then hit Enter or click "Ok."

![Enter "netplwiz" in the Run window.](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2023/11/1-4.png) 

 Select the user name you want to change, and then click "Properties."

![Select the user account you want to change, and then click "Properties."](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2022/02/click-name-then-click-properties.png)

 Whether you're using a Microsoft account or a local account, you should restart your PC to make sure the changes stick.

 It is also important to note that your user folder will not be renamed or moved. Windows and many other applications store files and settings in sub-folders of your user folder — consequently, it is difficult to change the name of your user folder without breaking things. So if your original user name was Bill, and you changed it to Tom, your user folder will still be "C:\\Users\\Bill" rather than "C:\\Users\\Tom ."
